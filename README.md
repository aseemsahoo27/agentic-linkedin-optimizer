# 🤖 Agentic LinkedIn Profile Optimizer

A multi-agent AI system that analyzes and improves LinkedIn profiles using iterative reasoning, evaluation loops, and memory.

---

## 🚀 Features

- Multi-agent architecture (Planner, Researcher, Analyst, Decision, Evaluator)
- LinkedIn profile ingestion
- Iterative improvement loop
- Context-aware evaluation
- Memory tracking of previous attempts

---

## 🧠 How It Works

1. LinkedIn Agent fetches profile data
2. Planner breaks task into steps
3. Researcher gathers structured data
4. Analyst improves content progressively
5. Decision Agent generates final output
6. Evaluator validates quality
7. System retries until output is strong

---

## 🔁 Example Flow

Attempt 1 → Rejected ❌  
Attempt 2 → Improved but weak ❌  
Attempt 3 → Strong output ✅
